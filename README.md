# Chat Web App

A simple web-based toy app for chatting

## How to use

```
$ npm i && npm start
```

And point your browser to http://localhost:5000. Optionally, specify a port by supplying the PORT env variable.

## Run

```
$ npm start
```

## Build

```
$ npm run build
```

## watch

```
$ npm run watch
```

## Develop

```
$ npm run dev
```

## Web
```
http://localhost:5000
```
